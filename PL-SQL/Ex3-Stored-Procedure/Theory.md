# Exercise 3 - Stored Procedures

## Objective
To implement stored procedures in Oracle PL/SQL for automating banking operations such as:
- Monthly interest calculation
- Employee bonus update
- Fund transfer between accounts

A stored procedure is a named PL/SQL block that is stored in the Oracle database and can be executed whenever required. Stored procedures improve code reusability, simplify complex database operations, and enhance performance by allowing business logic to execute directly within the database. They can accept parameters, perform calculations, update tables, and return results, making them useful for automating repetitive tasks such as interest calculation, salary updates, and fund transfers.
